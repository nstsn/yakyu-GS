# Knowledge Base (ナレッジベース)

本プロジェクト「NPB 満塁ホームラン後の得点動向検証」を通じて得られた知見、分析結果、および技術的な主要決定事項を記録します。

## 1. 検証結果・インサイト (Key Insights)

### 1.1 「満塁ホームラン後は打線が冷える」説の検証
- **結論**: **間違い (Myth Busted)**
- **データ**: 満塁弾の直後（残りイニング）における平均追加点は **約2.0点**。
- **比較**: 他のビッグイニング（4点以上）と比較しても、得点ペース（Run Rate）に有意な減少は見られない。むしろ、「一気に4点入った」勢いは維持される傾向にある。

### 1.2 イニング帯によるモメンタムの違い (Momentum by Stage)
得点ペースには、満塁弾が発生したイニング帯によって明確な傾向差が存在する。

| イニング帯 | 傾向 | 解釈 |
| :--- | :--- | :--- |
| **序盤・中盤 (1-6回)** | **High Momentum** | 「もっと点を取るぞ」という姿勢が続き、得点ペースが高い（Run Rate換算で1試合5点ペース以上）。 |
| **終盤 (7-9回)** | **Low Momentum** | 得点ペースが急激に低下する（Run Rate換算で1試合2点未満）。ここだけ「店じまい」現象が見られる（勝勢確定による攻撃の手控え等）。 |

## 2. 分析ロジック・定義 (Analysis Logic)

### 2.1 「以降得点」の定義
- 対象イベントが発生したイニングの **次** のイニングから、9回裏（または試合終了）までの総得点。
- 当該イニングに入った追加点は含まない（「満塁弾の *後*」の効果を純粋に測るため）。

### 2.2 特異値・NULLの扱い
- **サヨナラ勝ち/コールドゲーム**: 9回まで攻撃が行われない場合（NULL）、そのイニングは「攻撃機会」の分母から除外する。
- **以降得点率 (Post Run Rate)**: `以降得点の合計 / 残り攻撃イニング数の合計` で算出。
- **ゼロ埋め**: 攻撃機会があったが得点が入らなかった場合は `0` として扱うが、攻撃機会自体がなかった（9回裏なし等）場合は `NULL`（除外）とする。

## 3. データ処理のポイント
- **イベント抽出**: `rbi=4` かつ `hr=1` を基本とするが、塁状況 (`on_base`) の厳密なチェックも併用。
- **比較対象**: 単に「満塁弾以外」ではなく、「満塁弾以外で *同じく4点以上入ったイニング* (=ビッグイニング)」と比較することで、条件を揃えた検証を行っている。
